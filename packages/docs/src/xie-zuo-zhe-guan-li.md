# 🤝 协作者管理

## 管理协作者

右上角 **设置按钮** 展开，点击 **管理协作者**，可以通过复制链接链接分享给参与者

::: warning

修改参与者权限后，参与者刷新页面后生效。

:::

## 接受邀请

好友在收到邀请链接后打开如果已登录则直接获得旅程权限，未登录亦可在提示框中点击登录或注册后进入
