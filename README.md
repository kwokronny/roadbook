# 小肥路书帮助文档


![layout](packages/docs/src/public/layout.png)


多次旅行经验汇集的路书工具，让你在可视地图的工具下直观的快速安排自己的行程与在旅行中快速导航至目地的。


## [项目示例](https://roadbook.kwokronny.com) （注意：示例项目，请注意数据保存）


## 帮助文档
- [Github](https://kwokronny.github.io/roadbook)
  
## 特点 💡

- 简单易上手
- 响应式页面
- 可视化地图快速编排
- 支持 Docker 自建服务
